## Ejecutar este demo

```sh
node index.js
```

Una vez ejecutado este comando se iniciará el servidor Fastify en el puerto 3000 y al hacer solicitud a raíz verás los diferentes logs que se ejecutan por cada paso del ciclo de vida de Fastify.
